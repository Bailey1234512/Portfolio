The .triangle class is used to position the triangle element under the correct menu item on the navbar and gives the triangle the correct dimensions. The .triangle:before pseudo-element ensures the triangle's position and dimensions and also gives the triangle a background color of #67809f. The clip-path property is used to create the shape of the triangle. The clip-path: polygon(50% 70%, 0 0, 100% 0); is used to create a triangle shape using CSS, the first value is the point where the triangle will be starting, the second value is the point where the triangle will end and the third value is the point where the triangle will be ending. The clip-path property is used to clip an element to a custom shape. It is an experimental property which is not yet widely supported in all browsers. The position: absolute is used to position the triangle element at the top left corner of the parent element and display: block is used to set the display property of the element. The CSS property left: calc(100% / 7); was used to align all the triangles horizontally by positioning the second triangle at the width of the parent container divided by 7, the first triangle at 0% and the last triangle at the width of the parent container multiplied by 600% divided by 7. The top property of the nth-child selector was used to adjust the vertical position of the triangles in relation to their parent container to ensure they all align horizontally. If you wanted to change the height of the triangle you would have to change all of the n-th childs top by the height of the triangle multiplied by its position. So if it is the third triangle then it would be
0-(height * 3) Due to time limitations I couldn't find a way during the creation of the nav-bar to make the top calculation automated. But it could probably be done by instead of changing the height of the triangle you set the height of the triangle to a percentage of the height of the triangle container, which in this case is .triangle-row. So for a 60px contianer you could do height: calc(100%*0.6) to get a height of 60% which would be 60px I'm not sure how you could then store this value using css and then put that value as the top, but you could do this part with Javascript instead. When viewed on a small screen or using the browser's developer tools to reduce the screen size, the triangles may not scale properly.
